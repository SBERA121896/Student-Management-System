# Student-Management-System
A Python and MySQL based Student Management System for managing student records.

## Features
- Add Student
- View Student
- Update Student
- Delete Student
- Display All Students
- Input Validation
- MySQL Database Integration

## Technologies Used
- Python
- MySQL
